2. What is the difference between inline, internal, and external CSS?

Student's Answer:

Inline CSS applies to a single HTML element. Internal CSS is defined in the <head> section using <style> tags. External CSS is a separate .css file linked to the HTML.

3. Explain the Document Object Model (DOM) and its role in web development, including a simple JavaScript example to manipulate a DOM element.

Student's Answer:

The DOM represents the structure of an HTML or XML document as a tree of objects. It provides a programming interface that allows scripts like JavaScript to dynamically read, modify, and update the content and structure of a webpage. It is crucial for interactive web pages. For example, document.getElementById('myId').innerHTML = 'New Content' changes the text content of an element.
